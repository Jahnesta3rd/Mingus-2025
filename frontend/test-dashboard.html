🔧 API Integration

The dashboard integrates with the following API endpoints:

  • POST /api/auth/login - User authentication
  • GET /api/auth/verify - Token verification
  • POST /api/risk/assess-and-track - Risk analysis
  • GET /api/risk/dashboard-state - Dashboard state
  • POST /api/risk/trigger-recommendations - Job recommendations
  • POST /api/location/intelligence - Location analysis
  • POST /api/analytics/dashboard - Analytics data
  • POST /api/analytics/user-behavior/track-interaction - User tracking
  • GET /api/user/activity/recent - User activity
